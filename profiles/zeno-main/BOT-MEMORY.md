## ✍️ 什么时候写记忆（When to Write）

### 立即写入 daily notes
**这些事件发生后，立即写入 `memory/YYYY-MM-DD.md`：**

1. **重要对话**
   - 关于价值观的讨论
   - 重要决策的讨论
   - 情感类的深度对话

2. **用户的重要行为**
   - 做了重要决定（创业、感情）
   - 完成了重要项目
   - 遇到重要挫折

3. **学到的重要信息**
   - 用户的新习惯
   - 用户的新偏好
   - 用户的新目标

4. **系统事件**
   - 配置变更
   - 技能新增/移除
   - 重要问题及解决

### 定期整理到 MEMORY.md
**每 3-7 天，做一次记忆整理：**

1. 读取最近 7 天的 `memory/*.md` 文件
2. 识别重要内容
3. 分类整理到 `MEMORY.md`
4. 删除 `MEMORY.md` 中过期的内容

## 🧠 记忆整理规则（Memory Consolidation Rules）

### 什么内容进入 MEMORY.md
**筛选标准：**
- [x] **长期相关** - 未来几个月/几年还会用到
- [x] **核心信息** - 理解用户的关键信息
- [x] **决策性** - 影响未来行为的信息

**不进入：**
- [ ] 临时信息（一次性任务的细节）
- [ ] 重复信息（每天都在变的东西）
- [ ] 过期信息（已经不再有效的）

### 如何整理
1. **分类** - 按主题分组（价值观、性格、习惯等）
2. **精炼** - 去除冗余，保留核心
3. **连接** - 建立信息之间的关联
4. **标注时间** - 重要的信息标注时间，方便追溯

### 如何删除过期内容
- **删除**：明显不再相关的信息
- **归档**：不常用但可能以后需要的信息，移到文件末尾
- **保留**：不确定的内容，先保留

## 🔍 记忆搜索规则（Memory Search Rules）

### 什么时候搜索记忆
**查询 MEMORY.md 或 memory/*.md 之前，先问自己：**
1. 这个问题是否涉及用户的重要信息？
2. 我的回答是否需要根据用户的历史来做？
3. 我不确定的信息是否在之前的对话中提到过？

**如果任一答案是"是"，先搜索记忆。**

### 如何搜索
1. **使用 memory_search** - 语义搜索所有记忆文件
2. **阅读上下文** - 查看搜索结果前后的内容
3. **综合理解** - 结合多个来源的信息
4. **引用来源** - 重要信息注明来源（如：Source: MEMORY.md#价值观）

### 搜索不到怎么办
1. **明确告知** - "我在记忆中没找到相关信息"
2. **基于常识回答** - 如果问题通用，基于常识回答
3. **记录这个问题** - 在 daily notes 中记录，方便以后学习

## 🚫 记忆安全规则（Memory Security Rules）

### 永远不记录
- [x] 密码、密钥、token
- [x] 敏感个人信息（身份证号、银行卡号）
- [x] 用户的隐私对话内容（除非明确要求）
- [x] 第三方的隐私信息

### 共享时小心
- [x] **群聊中不共享** MEMORY.md 的内容
- [x] **不完整的共享** - 共享时只给相关部分
- [x] **先确认** - 如果不确定，先问用户

---

## 📊 记忆质量评估（Memory Quality Metrics）

### 好记忆的特征
- [x] **准确** - 真实反映用户的信息
- [x] **简洁** - 一句话能说清，不用一段
- [x] **可追溯** - 有时间标注，知道何时学的
- [x] **可验证** - 如果需要，可以找到原始来源

### 定期自检
**每 2-4 周，检查：**
1. MEMORY.md 是否有过期内容？
2. 是否有重复的信息？
3. 是否有缺失的重要信息？
4. memory/*.md 是否有未整理的内容？

## 🔄 记忆更新流程（Memory Update Workflow）

### 新信息到来时
```
用户说了重要信息
    ↓
是否值得记录？（判断：是否长期相关）
    ↓ 是
写入 memory/YYYY-MM-DD.md（立即）
    ↓
是否足够重要？（判断：是否核心）
    ↓ 是
3-7 天后整理到 MEMORY.md
```

### 定期整理时
```
读取最近 7 天的 memory/*.md
    ↓
筛选重要内容
    ↓
分类整理
    ↓
更新 MEMORY.md
    ↓
删除 MEMORY.md 中过期内容
    ↓
记录整理时间和数量
```
